Quick Start

Add this single line of code to your website, just before the closing </body> tag:

<script src="https://your-domain.com/widget.js" data-board-slug="your-board-slug"></script>

Replace your-domain.com with your ProductQuarry domain and your-board-slug with your actual board slug.

Configuration Options

Customize the widget behavior using data attributes:

Attribute Description Default
data-board-slug Your board slug (required) -
data-api-url API endpoint URL Current domain
data-position Widget position on page bottom-right
data-theme Color theme light
data-button-text Trigger button text Feedback

Position Options

  • bottom-right - Bottom right corner (default)
  • bottom-left - Bottom left corner
  • top-right - Top right corner
  • top-left - Top left corner

Theme Options

  • light - Light theme (default)
  • dark - Dark theme

Advanced Example

Here's an example with custom configuration:

<script
  src="https://your-domain.com/widget.js"
  data-board-slug="my-product"
  data-position="bottom-left"
  data-theme="dark"
  data-button-text="Send Feedback"
></script>
